Ex5_2:prime2(prime_number?)

前章の課題で求めた素数判定プログラムをmethodにせよ．駆動部は

a = gets.chomp.to_i
puts a
if prime?(a) then
  print  a,” is a prime number.\n”  
else
  print  a,” is not a prime number.\n”
end

つまり，素数ならtrue, なければfalseをprime_number?は返す．
